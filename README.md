# Project Overview

This repository contains my Digital Literacy Portfolio built as part of CSE0001.
I worked through five tasks covering different areas of digital literacy — from designing an infographic to  understanding
cybercrime. 
The goal was to build practical digital skills and document them in one place.

# Module Summaries

Task 1 – Infographics
Created a one page Digital Literacy Awareness Infographic 
using Canva covering what digital literacy means, useful 
tools for students, and safe internet practices.

Task 2 – Digital Portfolio
Set up professional profiles on GitHub, LinkedIn and Kaggle 
with relevant academic information and a GitHub profile README.

Task 3 – Coding and Collaboration Platforms
Solved two beginner Python challenges on HackerRank and 
created a five question Digital Literacy Awareness Quiz 
using Google Forms.

Task 4 – Email Etiquette
Drafted two professional emails and created a Social Media 
Do's and Don'ts checklist for college students.

Task 5 – Cybercrime Awareness
Wrote a case study on UPI/Online Payment Fraud and created 
a Stay Safe Online prevention checklist with reporting 
channels for Indian students.
